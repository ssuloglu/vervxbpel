The application is used to transform from source model, VxBPEL to target model, FTS model.
Transformation rules are given in a seperate file, Transformation Rules. The transformation is implemented in a single Python file.
File name: tool4transformvxbpel2fts.py
@ Input: VxBPEL variable orchestration
@ Output: Feature model of variable orchestration in TVL specification and fPromela model
